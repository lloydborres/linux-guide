# Steam on Linux

Check for compatibility for games in [ProtonDB](https://www.protondb.com)

Optionally install [GE-Proton](https://github.com/GloriousEggroll/proton-ge-custom)

## Game Configurations

### Final Fantasy VII Rebirth

**Compatibility:** default

**Launch Options:**

```
VKD3D_FEATURE_LEVEL=12_2 VKD3D_SHADER_MODEL=6_6 %command%
```

### Wuthering Waves

**Compatibility:** default

**Launch Options:**

```
SteamOS=1 %commad%
```
